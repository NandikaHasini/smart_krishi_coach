# Smart Krishi Coach

Smart Krishi Coach is a web application that helps farmers choose crops using predefined agricultural rules.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## How to Run
1. Install Python
2. Run: pip install -r requirements.txt
3. Run: python app.py
4. Open browser and go to http://127.0.0.1:5000/
